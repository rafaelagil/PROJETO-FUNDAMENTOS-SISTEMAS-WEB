# 🐾 PetShop Cão & Gato

## 📌 Descrição
Este projeto consiste em um site básico para uma pet shop, desenvolvido utilizando apenas HTML puro.  

O objetivo principal foi criar a estrutura inicial de um site, com navegação entre páginas e exibição de conteúdo visual, simulando um ambiente real de um negócio de pet shop.

---

## 🚀 Funcionalidades
- Navegação entre páginas
- Exibição de imagens de produtos/serviços
- Estrutura básica de um site institucional
- Organização de conteúdo em múltiplas páginas

---

## 🛠️ Tecnologias utilizadas
- HTML5

---

## 📂 Estrutura do projeto
O projeto está organizado em três arquivos principais e uma pasta com as imagens dos produtos e serviços.

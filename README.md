# 🐾 PetShop Cão & Gato

## 📌 Descrição

Este projeto consiste em um site para uma pet shop, desenvolvido com HTML, CSS e JavaScript.

Inicialmente criado com HTML puro, o projeto evoluiu para incluir interatividade e organização mais profissional, simulando um sistema real de um pet shop com área de cliente e área administrativa.

O sistema permite:

* Navegação entre páginas
* Visualização de produtos e serviços
* Cadastro de clientes e pets
* Agendamento de serviços
* Gerenciamento de agendamentos (admin)

---

## 🚀 Funcionalidades

### 👤 Área do cliente

* Navegação entre páginas
* Visualização de produtos e serviços
* Cadastro de cliente
* Cadastro de pet
* Agendamento de serviços

### 🔐 Área administrativa

* Login de administrador
* Proteção de acesso à página de agendamentos
* Listagem de agendamentos
* Exclusão de agendamentos

### ⚙️ Sistema

* Armazenamento de dados com LocalStorage
* Manipulação dinâmica com JavaScript
* Interface com Bootstrap (responsiva)
* Organização de código com script centralizado

---

## 🛠️ Tecnologias utilizadas

* HTML5

* Bootstrap 5
* JavaScript (puro)

## ⚠️ Limitações atuais

* Não possui backend (funciona apenas no navegador)
* Dados armazenados em LocalStorage (podem ser apagados)
* Login simples sem criptografia
* Não possui banco de dados real
* Não possui edição de agendamentos (apenas exclusão)
